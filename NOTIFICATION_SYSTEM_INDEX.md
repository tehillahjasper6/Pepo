## üìä Features Overview

The notification system provides:

1. **Follow-Based Notifications** ‚úÖ
   - Users get notified when NGOs they follow post items
   - Automatic follower discovery and notification

2. **User Preferences** ‚úÖ
   - Global notification control
   - Per-NGO notification control
   - Preference hierarchy

3. **Queue-Based Processing** ‚úÖ
   - Non-blocking API requests
   - Background worker processes notifications
   - Batch processing for scalability

4. **Background Worker** ‚úÖ
   - Scheduled cron jobs
   - Automatic retry mechanism
   - Cleanup of old records

5. **In-App & Push Notifications** ‚úÖ
   - Database-backed in-app notifications
   - Push notifications to devices
   - Async delivery
